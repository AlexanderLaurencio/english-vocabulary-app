This is an English Vocabulary App that allows you store English words (actually any word from any language) in form of cards. 

Functionalities

-Dark and light mode
-Searchbar (that really works) 
-filters (to filter words by their type) 

In addition to this, the website is RESPONSIVE, so it looks good in smartphones, tablets and computers.

Technologies used:
-HTML
-CSS
-JAVASCRIPT
-REACT.JS
